# Origin Statement / 项目起源声明

Huiyin / 回音 was originally proposed and initially implemented by r1589781959-cloud in 2026.

回音是一个实验性的情感陪伴类 AI Agent 项目，最初由 r1589781959-cloud 提出并实现。

## Original Direction / 原始方向

项目最初关注以下方向：

- 自然、连续的陪伴式对话
- 长期记忆与历史消息检索
- 多模型接入
- 本地优先的数据存储
- 主动消息与轻量陪伴
- 面向个人使用的可扩展 Agent 架构
- 面向陪伴体验的上下文连续性
- 面向安全边界的回复控制
